# Clause 5.4(9) & 5.5 – Secondary Dwelling Provisions

**PP Number:** PP-2025-2534  
**Council:** Wagga Wagga  
**Address:** []  
**Exhibition:** 2026-03-30 to 2026-05-01  
**Type:** council-led

## What's being proposed

Wagga Wagga City Council is proposing to amend the Wagga Wagga Local Environmental Plan 2010 to change a rule about secondary dwellings across the entire Local Government Area [doc: LEP25.0001 - draft Planning Proposal - For submission | p.6]. The proposal seeks to increase the maximum allowable size of a secondary dwelling from 33% of the main house’s floor area to 65% [doc: LEP25.0001 - draft Planning Proposal - For submission | p.4][doc: LEP25.0001 - draft Planning Proposal - For submission | p.5][doc: LEP25.0001 - draft Planning Proposal - For submission | p.11]. This change would apply to properties in both rural and non-rural zones [doc: LEP25.0001 - draft Planning Proposal - For submission | p.4][doc: LEP25.0001 - draft Planning Proposal - For submission | p.5][doc: LEP25.0001 - draft Planning Proposal - For submission | p.10].

The main objective is to provide more diverse and affordable housing options in Wagga Wagga, including 1–2 bedroom accommodation, by allowing larger and more practical secondary dwellings to be built [doc: LEP25.0001 - draft Planning Proposal - For submission | p.11][doc: LEP25.0001 - draft Planning Proposal - For submission | p.13][doc: LEP25.0001 - draft Planning Proposal - For submission | p.43]. This is intended to encourage efficient development using existing infrastructure and to support families or residents looking to age in place [doc: LEP25.0001 - draft Planning Proposal - For submission | p.11][doc: LEP25.0001 - draft Planning Proposal - For submission | p.19][doc: LEP25.0001 - draft Planning Proposal - For submission | p.43]. The proposal ensures that secondary dwellings will maintain their role as smaller, additional homes on a property, remaining clearly subordinate to the main dwelling [doc: LEP25.0001 - draft Planning Proposal - For submission | p.13][doc: LEP25.0001 - draft Planning Proposal - For submission | p.43]. No changes to land zoning maps are required [doc: LEP25.0001 - draft Planning Proposal - For submission | p.10][doc: LEP25.0001 - draft Planning Proposal - For submission | p.11][doc: LEP25.0001 - draft Planning Proposal - For submission | p.40].

## What changes on the ground

This proposal does not change any land zoning or require new maps [doc: LEP25.0001 - draft Planning Proposal - For submission | p.10][doc: LEP25.0001 - draft Planning Proposal - For submission | p.40]. It applies to all properties in the Wagga Wagga Local Government Area where secondary dwellings are currently allowed, including R1 General Residential, R3 Medium Density Residential, R5 Large Lot Residential, MU1 Mixed Use, C4 Environmental Living, RU1 Primary Production, RU2 Rural Landscape, RU4 Primary Production Small Lots, and RU5 Village zones [doc: LEP25.0001 - draft Planning Proposal - For submission | p.10].

The main change is to the maximum size of secondary dwellings. For properties in non-rural zones, the maximum floor area for a secondary dwelling will increase from 33% to 65% of the main house's floor area [doc: LEP25.0001 - draft Planning Proposal - For submission | p.5][doc: LEP25.0001 - draft Planning Proposal - For submission | p.11]. Similarly, for properties in rural zones, the maximum floor area for a secondary dwelling will also increase from 33% to 65% of the main house's floor area [doc: LEP25.0001 - draft Planning Proposal - For submission | p.5][doc: LEP25.0001 - draft Planning Proposal - For submission | p.11].

There are no changes to height limits, site coverage, setbacks, lot sizes, or other existing development controls [doc: LEP25.0001 - draft Planning Proposal - For submission | p.38]. The proposal is not expected to significantly increase demand for services or new infrastructure, aiming to use existing facilities [doc: LEP25.0001 - draft Planning Proposal - For submission | p.5][doc: LEP25.0001 - draft Planning Proposal - For submission | p.11].
