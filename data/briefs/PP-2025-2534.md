# Clause 5.4(9) & 5.5 – Secondary Dwelling Provisions

**PP Number:** PP-2025-2534  
**Council:** Wagga Wagga  
**Address:** []  
**Exhibition:** 2026-03-30 to 2026-05-01  
**Type:** council-led

## What's being proposed

Wagga Wagga City Council is proposing to change a rule in the Wagga Wagga Local Environmental Plan 2010 that affects secondary dwellings across the entire local government area [doc: LEP25.0001 - draft Planning Proposal - For submission | p.6].

The main proposal is to increase the maximum allowed size for a secondary dwelling [doc: LEP25.0001 - draft Planning Proposal - For submission | p.11]. Currently, a secondary dwelling can be up to 33% of the size of the main house on the property [doc: LEP25.0001 - draft Planning Proposal - For submission | p.4]. The proposal would change this to allow secondary dwellings to be up to 65% of the size of the main house [doc: LEP25.0001 - draft Planning Proposal - For submission | p.4]. This change would apply to secondary dwellings on all types of land where they are currently permitted, including rural zones and other residential areas [doc: LEP25.0001 - draft Planning Proposal - For submission | p.4][doc: LEP25.0001 - draft Planning Proposal - For submission | p.5].

The aim is to offer more diverse housing choices by allowing larger and more practical secondary dwellings [doc: LEP25.0001 - draft Planning Proposal - For submission | p.11][doc: LEP25.0001 - draft Planning Proposal - For submission | p.13][doc: LEP25.0001 - draft Planning Proposal - For submission | p.43]. This is intended to encourage more affordable housing options, provide opportunities for younger people, single-person households, retirees, and multi-generational families, and make better use of existing services and infrastructure [doc: LEP25.0001 - draft Planning Proposal - For submission | p.7][doc: LEP25.0001 - draft Planning Proposal - For submission | p.11][doc: LEP25.0001 - draft Planning Proposal - For submission | p.13][doc: LEP25.0001 - draft Planning Proposal - For submission | p.43].

## What changes on the ground

This proposal changes the rules for how large secondary dwellings (like granny flats) can be built across the Wagga Wagga Local Government Area [doc: LEP25.0001 - draft Planning Proposal - For submission | p.10].

Currently, a secondary dwelling can be no more than 33% of the size of the main house [doc: LEP25.0001 - draft Planning Proposal - For submission | p.4][doc: LEP25.0001 - draft Planning Proposal - For submission | p.5]. The change will allow secondary dwellings to be up to 65% of the main house's floor area [doc: LEP25.0001 - draft Planning Proposal - For submission | p.4][doc: LEP25.0001 - draft Planning Proposal - For submission | p.5]. This applies to both non-rural zones (R1 General Residential, R3 Medium Density Residential, R5 Large Lot Residential, MU1 Mixed Use, C4 Environmental Living) and rural zones (RU1 Primary Production, RU2 Rural Landscape, RU4 Primary Production Small Lots, RU5 Village) [doc: LEP25.0001 - draft Planning Proposal - For submission | p.10][doc: LEP25.0001 - draft Planning Proposal - For submission | p.5].

No land will be rezoned as part of this proposal [doc: LEP25.0001 - draft Planning Proposal - For submission | p.10][doc: LEP25.0001 - draft Planning Proposal - For submission | p.38]. It also does not change existing controls for building height, site coverage, setbacks, or other environmental assessment requirements [doc: LEP25.0001 - draft Planning Proposal - For submission | p.38]. There are no proposed amendments to planning maps [doc: LEP25.0001 - draft Planning Proposal - For submission | p.40][doc: LEP25.0001 - draft Planning Proposal - For submission | p.11]. The proposal is not expected to significantly increase demand for services or infrastructure [doc: LEP25.0001 - draft Planning Proposal - For submission | p.5].
