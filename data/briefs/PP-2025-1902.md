# Rezoning from R2 to RE2 Private Recreation at 55 Cabramatta Road East, Cabramatta (Cabra Bowls Club)

**PP Number:** PP-2025-1902  
**Council:** Not specified  
**Address:** ["55 Cabramatta Road East", "Cabramatta"]  
**Exhibition:** 2026-04-13 to 2026-05-22  
**Type:** council-led

## What's being proposed

The Cabramatta Bowling & Recreation Club is proposing to change the planning rules for the land at 55 Cabramatta Road East, Cabramatta [doc: Gateway request - Planning Proposal document | p.4][doc: Planning_Proposal_55_Cabramatta_Road_Cabramatta | p.3]. Currently, this site is zoned for R2 Low Density Residential [doc: Gateway request - Planning Proposal document | p.4]. The proposal seeks to rezone it to RE2 Private Recreation, which is the same zone as the adjoining Cabramatta Bowling Club [doc: Gateway request - Planning Proposal document | p.4].

The main aim of this change is to allow for a new, privately owned carpark to be built on the property [doc: Gateway request - Planning Proposal document | p.4]. The existing single-storey house and garage at 55 Cabramatta Road East would be demolished [doc: Planning_Proposal_55_Cabramatta_Road_Cabramatta | p.3]. In their place, an at-grade, sealed car park for approximately 45 to 50 vehicles, along with boundary landscaping, would be constructed [doc: Planning_Proposal_55_Cabramatta_Road_Cabramatta | p.3]. This new carpark would connect to and operate with the Cabramatta Bowling Club at 82 Longfield Street, Cabramatta, providing more on-site parking for club patrons [doc: Gateway request - Planning Proposal document | p.4][doc: Planning_Proposal_55_Cabramatta_Road_Cabramatta | p.3]. The current vehicle access from Cabramatta Road would be removed [doc: Planning_Proposal_55_Cabramatta_Road_Cabramatta | p.3].

This proposal is a "map-only" change to the Fairfield Local Environmental Plan 2013 [doc: Gateway request - Planning Proposal document | p.4]. Building the carpark itself would require a separate development application to be considered by the Council in the future [doc: Gateway request - Planning Proposal document | p.4].

## What changes on the ground

The proposal will change the planning rules for the property at 55 Cabramatta Road East, Cabramatta [doc: Gateway request - Planning Proposal document | p.6]. Currently, the land is zoned R2 Low Density Residential [doc: Gateway request - Planning Proposal document | p.6]. This plan proposes to rezone it to RE2 Private Recreation [doc: Gateway request - Planning Proposal document | p.6].

These changes will remove several building and lot size limits. The maximum height of buildings, currently set at 9 metres, will be removed, meaning there will be no height control [doc: Gateway request - Planning Proposal document | p.6]. The Floor Space Ratio (FSR) control of 0.45:1 will also be removed [doc: Gateway request - Planning Proposal document | p.6].

Furthermore, the minimum lot size requirement of 450 square metres will be removed, so there will be no minimum lot size control [doc: Gateway request - Planning Proposal document | p.6]. For dual occupancy developments, the minimum lot size of 900 square metres will also no longer apply [doc: Gateway request - Planning Proposal document | p.6]. These amendments aim to make the site consistent with the adjoining RE2 Private Recreation zone of the Cabra Bowls Club at 82 Longfield Street, Cabramatta, to facilitate future car parking [doc: Gateway request - Planning Proposal document | p.26][doc: Gateway request - Planning Proposal document | p.6].
