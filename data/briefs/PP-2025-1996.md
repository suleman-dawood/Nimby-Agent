# Planning Proposal to Reclassify Community Land as Operational Land at 5A Waldron Street, Sandringham

**PP Number:** PP-2025-1996  
**Council:** Not specified  
**Address:** ["5A Waldron Street Sandringham 2219"]  
**Exhibition:** 2026-04-07 to 2026-05-05  
**Type:** council-led

## What's being proposed

This planning proposal is about changing how a piece of Bayside Council-owned land at 5A Waldron Street, Sandringham [doc: Planning Proposal Report | p.21], known as Zealander Walk [doc: Planning Proposal Report | p.7], is managed. The main idea is to reclassify this land from "community" land to "operational" land [doc: Planning Proposal Report | p.21][doc: Planning Proposal Report | p.7]. It is not considered a "public reserve" [doc: Planning Proposal Report | p.18]. This reclassification is an administrative change [doc: Planning Proposal Report | p.38] that would allow Bayside Council to potentially sell the land [doc: Planning Proposal Report | p.21][doc: Planning Proposal Report | p.7]. The goal is to facilitate the sale or redevelopment of this underutilised parcel [doc: Planning Proposal Report | p.18][doc: Planning Proposal Report | p.40]. This proposal was prepared by BMA Urban on behalf of Joe Gittany [doc: Planning Proposal Report | p.7].

## What changes on the ground

The proposal is about changing the classification of the land at 5A Waldron Street, Sandringham (Lot 79 in DP 14165), from “community” land to “operational” land [doc: Planning Proposal Report | p.5].

However, this specific planning proposal does not introduce any changes to the existing building rules for the site [doc: Planning Proposal Report | p.8][doc: Planning Proposal Report | p.16][doc: Planning Proposal Report | p.40]. The land will continue to be zoned R2 – Low Density Residential [doc: Planning Proposal Report | p.5][doc: Planning Proposal Report | p.16]. The maximum height for buildings will remain at 8.5 metres [doc: Planning Proposal Report | p.5], and the Floor Space Ratio (FSR), which limits the total floor area of buildings, will stay at 0.5:1 [doc: Planning Proposal Report | p.5]. No changes are proposed to the specific uses that are allowed or not allowed in the R2 zone [doc: Planning Proposal Report | p.16].

This proposal does not involve any new infrastructure [doc: Planning Proposal Report | p.40].
